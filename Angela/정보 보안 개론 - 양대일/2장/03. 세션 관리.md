## 지속적 인증 : (예시) 한 사용자가 인증 성공 -> 같은 아이디로 다시 한 번 인증 성공 ----> 다시 인증에 성공한 사용자가 원래의 사람인지 확인해야 함 -> 지속적 인증 필요
 - 보통 타임아웃을 설정해 일정 시간의 유휴 상태 이후 재인증 요구
 - 예 : 윈도우의 화면 보호기
 - 예 : 인터넷 뱅킹에서 지속적으로 패스워드를 물어보는 것, 패스워드 재설정
